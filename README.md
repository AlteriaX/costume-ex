# How to use:

Weapon glow: 
```
Format - weapon1 <number here (1-15)>

Example - weapon1 15
```
Dye: 
```
Format - dye1 R G B Z

Example - dye1 255 50 50 10

Type undye1 to remove.
```

Costumes: 
```
Format - use1 <type> <id>

Types - weapon, costume, head, face, back, footstep

Example - use1 costume 185406
```

Type ``reset1`` to restore your default looks.
